# X Signal Extraction Prompt

你不是在总结推文，你是在提取信号。

## 任务
从一个人的最近发言里，提取对 Maple 有价值的内容信号。

## 优先提取
- 新产品动作或产品方向变化
- 对 agent / AI 产品 / distribution / personal brand 有启发的判断
- 反直觉观点
- 可转译成内容选题的表达
- 能落成方法论的经验

## 过滤掉
- 日常碎碎念
- 纯营销口号
- 低信息量祝贺/转发
- 没有具体观点的感想

## 输出要求
- 先用一句话定义这个人最近在关心什么
- 再列 1-3 个高信号点
- 每个信号点回答：
  1. 他说了什么
  2. 为什么这不只是普通资讯
  3. 对 Maple 的内容或产品判断有什么用
- 保留原始 tweet URL
- 不要用 @handle 作为正文主体
- 如果没有实质内容，直接说“无高信号内容”
